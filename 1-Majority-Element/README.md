# Majority Element

## Problem

Given an array `nums` of size `n` return the _majority element_

When an element appears more than `[n / 2]` times in an array its called _majority element_.

## Example

```
Input: nums = [3,2,3]
Output: 3
```

```
Input: nums = [2,2,1,1,1,2,2]
Output: 2
```

## Constraints:

```
n === nums.length
1 <= n <= 5 * 10^4
-2^31 <= nums[i] <= 2^31 -1
```
